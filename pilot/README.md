## Pilot: PhosOpt end-to-end (train -> save -> optimize MNIST)

This folder contains a minimal, reproducible pipeline that:

- trains **PhosOpt** (shared encoder) on generated targets
- saves the trained model checkpoint
- evaluates/optimizes on a small subset of **MNIST letter** targets (EMNIST phosphene maps)

### Quickstart

From the repo root:

```bash
python pilot/run_phosopt_pilot.py --output_dir pilot/results --seed 42 --mnist_n 5
```

### What it produces

- `pilot/results/`
  - `phosopt_seed{seed}.pt` (saved model state_dict)
  - `pilot_results.jsonl` (per-target logs: zero-shot and fine-tune)
  - `pilot_summary.json` (aggregated metrics)

### Notes

- All code/comments are in English by project convention.
- The MNIST targets are the `mnist_letter_*.npy` maps generated by `code/targets/generator.py`.
